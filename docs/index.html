원천청년부 앱 개인정보처리방침

최종 수정일: 2026년 4월 6일

원천교회 청년부(이하 "운영자")는 「개인정보 보호법」에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하게 처리할 수 있도록 다음과 같이 개인정보처리방침을 수립·공개합니다.

1. 수집하는 개인정보 항목

수집 항목수집 목적필수 여부
이름회원 식별 및 로그인필수
비밀번호 (암호화 저장)로그인 인증필수
생년월일회원 관리선택
성별회원 관리선택
디바이스 토큰푸시 알림 발송선택

2. 개인정보의 수집 방법

  • 앱 회원 등록 시 관리자가 이름, 생년월일, 성별을 등록
  • 앱 최초 로그인 시 비밀번호 변경
  • 푸시 알림 권한 동의 시 디바이스 토큰 자동 수집

3. 개인정보의 이용 목적

  • 회원 인증 및 로그인 서비스 제공
  • 중보기도 작성·조회·관리 서비스 제공
  • 출결 관리 서비스 제공
  • 주간 푸시 알림 발송 (중보기도 안내)
  • 서비스 운영 및 개선

4. 개인정보의 보관 및 파기

  • 보관 기간: 회원 탈퇴(계정 삭제) 시까지
  • 파기 방법: 계정 삭제 요청 시 모든 개인정보 및 작성 콘텐츠를 즉시 삭제
  • 비밀번호는 bcrypt 단방향 암호화로 저장되며, 원문 복원이 불가합니다.

5. 개인정보의 제3자 제공

운영자는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 법령에 의한 요청이 있거나 이용자가 사전에 동의한 경우는 예외합니다.

6. 개인정보의 안전성 확보 조치

  • 비밀번호 bcrypt 암호화
  • JWT 기반 인증 토큰 사용
  • HTTPS 통신 (AWS API Gateway)
  • 데이터베이스 접근 제어 (AWS IAM)

7. 이용자의 권리

  • 계정 삭제: 앱 내 설정 > 계정 삭제를 통해 모든 개인정보 및 콘텐츠 삭제
  • 개인정보 열람: 앱 내 설정에서 본인 정보 확인
  • 문의: 아래 연락처를 통해 개인정보 관련 문의

8. 콘텐츠 신고

이용자는 부적절한 콘텐츠를 앱 내 신고 기능을 통해 신고할 수 있으며, 운영자는 검토 후 적절한 조치를 취합니다.
